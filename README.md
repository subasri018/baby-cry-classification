# baby-cry-classification
Real-time infant cry classification using audio processing, machine learning, and deep learning.
Project Overview:
This project is a real-time infant cry classification system built using Machine Learning and Deep Learning techniques. It captures live audio, processes it, and predicts the baby's needs.

Features:

🎤 Live audio recording
📊 Feature extraction using MFCC & Spectrogram
🤖 Machine Learning (SVM)
🧠 Deep Learning (CNN)
📷 Camera-based real-time display

Technologies Used:
Python
Librosa
Scikit-learn
TensorFlow
OpenCV

Note:
Due to limited dataset and variability in baby sounds, predictions may not always be accurate.
